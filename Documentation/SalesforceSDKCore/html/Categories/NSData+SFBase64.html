NSData(SFBase64) Category Reference

Declared in NSData+SFAdditions.h

Overview

Extension to NSData class to provide common functions. Added functionality includes: –base64 encoding of an NSData object –MD5 version of an NSData object –Gzip deflated representation of a gzip-compressed NSData object –Hex version of an NSData object
