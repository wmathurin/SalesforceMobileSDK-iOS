Overview

This class manages manages the inactivity timer. See NSTimer in the Objective-C documentation for more details.

+ registerTimer:target:selector:timerInterval:

  • Register the timer.
+ (void)registerTimer:(NSString *)timerName target:(id)target selector:(SEL)aSelector timerInterval:(NSTimeInterval)interval

Parameters

timerName

Name to use to register the timer.
target

The object to which to send the message specified by aSelector when the timer fires. See NSTimer documentation.
aSelector

The message to send to target when the timer fires. See NSTimer documentation.
interval

For a repeating timer, this parameter contains the number of seconds between firings of the timer. See NSTimer documentation.
