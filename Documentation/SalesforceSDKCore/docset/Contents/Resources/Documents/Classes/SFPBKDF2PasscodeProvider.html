Overview

Contains configuration values for generating an encryption key.

  saltLengthInBytes

The length in bytes to use for the random-generated salt. This value defaults to kSFPBKDFDefaultSaltByteLength in SFSDKCryptoUtils.h. NOTE: Differences in this value affect the generated key. You must keep track of this value if you wish to generate the same key at a future date.
